## 速记

- 商：`(quot 22 7)` -> `3`
- 余数：`(rem 22 7)` -> `1`
- 数字后面加大写 `M` 就是 `BigDecimal`
- 数字后面加大写 `N` 就是 `BigInt`
- 调用 Java 方法，如 `(.toUpperCase "Hello")` -> `"HELLO"`
- 空列表不为 false
- 零不为 false
