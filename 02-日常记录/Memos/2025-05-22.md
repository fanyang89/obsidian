## 检查漏洞是否修复

- CVE-2017-5645 Apache Log4j 代码问题漏洞
  - log4j < 2.8.2
- CVE-2019-17571 Apache Log4j不可信数据反序列化漏洞
  - This affects Log4j versions up to 1.2 up to 1.2.17.
- CVE-2021-44228 Apache Log4j 远程命令执行漏洞
  - Apache Log4j 2.0-beta9至2.14.1
- CVE-2021-45046 Apache Log4j 远程代码执行漏洞
  - 在 >= 2.0-beta9 且 <= 2.14.1
- CVE-2022-23305 Apache软件基金会 Apache Log4j 1.x等产品SQL注入漏洞
  - Apache Log4j 1.2 已于 2015 年 8 月结束生命周期。用户应升级到 Log4j 2
